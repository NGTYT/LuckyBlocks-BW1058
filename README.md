# LuckyBlocks-BW1058
LuckyBlocks is a fun Bedwars1058 addon that adds surprise to every game. Breaking lucky blocks can give items like Sleep Dust, Ice Bridge, Bridge Egg, and Inventory Copier. Every match becomes more exciting, unpredictable, and fun to play with friends.

# 🎲 LuckyBlocks

A fun and chaotic Minecraft plugin that brings unpredictable lucky block action to your games.  
Every match becomes exciting with surprise drops, traps, and special items.

## ✨ Features

Break lucky blocks and get random effects like:

- 😴 Sleep Dust
- 🚀 Player Launcher
- 📦 Inventory Copier
- ⚡ Lightning Scroll
- 🥚 Bridge Egg
- 🔥 Fireball
- 💣 TNT
- 🧊 Ice Bridge
- 🌈 Rainbow Wool

Every game is different, fast-paced, and full of surprises!

## 💬 Commands

/luckyblock  
Main command for all plugin features.

---

/luckyblock autosetup <arena> <cooldown>  
Automatically sets up lucky block generators for an arena.

---

/luckyblock addgenerator <arena> <team-name> <cooldown>  
Adds a lucky block generator for a specific team.

---

/luckyblock removegenerator <arena> <id>  
Removes a generator using its ID from data.yml.

---

## 🎮 Gameplay
